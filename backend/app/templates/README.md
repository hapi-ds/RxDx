# Document Templates

This directory contains templates for document generation.

## Template Types

### Word Templates (.docx)
- `default_invoice.docx` - Default invoice template with placeholders

### Template Variables

#### Invoice Template Variables
- `{{ invoice_number }}` - Invoice number
- `{{ invoice_date }}` - Invoice date
- `{{ billing_period }}` - Billing period string
- `{{ client_name }}` - Client name
- `{{ client_address }}` - Client address
- `{{ project_id }}` - Project ID
- `{{ line_items }}` - List of line items (use `{% for item in line_items %}`)
  - `{{ item.description }}` - Line item description
  - `{{ item.hours }}` - Hours worked
  - `{{ item.rate }}` - Hourly rate
  - `{{ item.subtotal }}` - Line item subtotal
- `{{ subtotal }}` - Invoice subtotal
- `{{ tax_rate }}` - Tax rate percentage
- `{{ tax_amount }}` - Tax amount
- `{{ total }}` - Total amount
- `{{ currency }}` - Currency code
- `{{ notes }}` - Additional notes
- `{{ generated_by }}` - Name of user who generated the invoice

## Creating Custom Templates

1. Create a new Word document (.docx)
2. Add placeholders using Jinja2 syntax: `{{ variable_name }}`
3. For loops, use: `{% for item in items %}...{% endfor %}`
4. Save the template in this directory
5. Reference the template name (without .docx) in the API request

## Example Invoice Template Structure

```
INVOICE

Invoice Number: {{ invoice_number }}
Date: {{ invoice_date }}
Billing Period: {{ billing_period }}

Bill To:
{{ client_name }}
{{ client_address }}

Project: {{ project_id }}

| Description | Hours | Rate | Amount |
|-------------|-------|------|--------|
{% for item in line_items %}
| {{ item.description }} | {{ item.hours }} | {{ item.rate }} | {{ item.subtotal }} |
{% endfor %}

Subtotal: {{ currency }} {{ subtotal }}
Tax ({{ tax_rate }}): {{ currency }} {{ tax_amount }}
Total: {{ currency }} {{ total }}

Notes: {{ notes }}

Generated by: {{ generated_by }}
```
